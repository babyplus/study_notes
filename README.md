# 零售管理 

## 零售出库  

用于散户使用，不能欠款。

## 零售退货  

用于零售出库单据的退货。

# 采购管理  

## 采购订单 

用于对采购订单进行审核，不涉及付款金额。 

## 采购入库  

用于登记入库信息，可关联采购订单或单独创建；可以使用多个账户进行付款。

## 采购退货  

用于采购入库单据的退货。

# 销售管理  

## 销售订单  

用于对销售订单进行审核，不涉及付款金额。 

## 销售出库  

用于销售出库信息，可关联销售订单或单独创建；可以使用多个账户进行收款。

## 销售退货  

用于销售出库单据的退货。

# 仓库管理  

## 其它入库  

库存初始化，生产管理模块的成品入库（猜测是个收费模块）。

## 其它出库  

库存初始化，生产管理模块的领料出库（猜测是个收费模块）。

## 调拨出库  

用于两个仓库之间进行调拨，调拨单会影响库存。

## 组装单  

用于将多个商品合并成一个商品，被合并的商品库存减少，合并后的商品库存增加。

## 拆卸单  

用于将一个商品拆分成多个商品，被拆分的商品库存减少，拆分后的商品库存增加。

# 财务管理  

## 收入单  

主要用于记录销售收入外的收入。

## 支出单  

主要用于记录采购支出外的支出。

## 收款单  

收款单所收金额只对付款单位的应收应付产生影响。收款单的优惠金额会对利润产生影响，但不影响付款单位的应收应付。

## 付款单  

付款单所收金额只对收款单位的应收应付产生影响。

## 转账单  

从一个银行存款账户转入另一个银行存款账户。

## 收预付款  

针对会员模块，收取预付款。（猜测是个付费模块）
